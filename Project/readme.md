# DRINKS PROYECT 
Clone the proyect 
 `git clone https://github.com/fgorgojo/IdentityAuthorization.git`

## Starting the proyect in Windows environment

### back-end
 - Environment variables
 ```
   $env:FLASK_APP="src.api"
   $env:FLASK_DEBUG="true"
   $env:AUTH0_DOMAIN="dev-czgv2o0xdmpmx1qb.us.auth0.com"
   $env:ALGORITHMS="RS256"
   $env:API_AUDIENCE="drinks"
 ```

 - Navigate to the folder backend: 
   ` cd .\Project\03_coffee_shop_full_stack\starter_code\backend`

 - Start back-end 
   `flask run`


### Start front end 
 - Naviage to the folder fronted:
  `cd .\Project\03_coffee_shop_full_stack\starter_code\backend`

 - Start front end
 `ionic serve`
